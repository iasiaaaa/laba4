# Лабораторна робота №4 - Рефакторинг коду

## Порівняння продуктивності

| Версія | Час виконання | Примітки |
|--------|---------------|----------|
| До рефакторингу 1 поток | ~1485 мс | Базова версія |
| Після рефакторингу 1 поток | ~1491 мс | Збільшення на ~6 мс |
| До рефакторингу 8 потоків | ~241 мс | Базова версія |
| Після рефакторингу 8 потоків | ~248 мс | Збільшення на ~7 мс |

**Висновок:** Незначне збільшення часу виконання (~4-8 мс) є прийнятним компромісом за покращення якості коду.
